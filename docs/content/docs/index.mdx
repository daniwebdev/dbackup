# Welcome to DBackup

**DBackup** is a robust, high-performance database backup utility written in **Rust**, designed for reliability and ease of use through YAML-based configuration.

Whether you're backing up a small development database or managing multiple production systems, DBackup provides a flexible, efficient, and straightforward solution for your backup needs.

## Why DBackup?

- **Simple Configuration**: YAML-based setup - no complex scripts to write
- **High Performance**: Parallel backup mode for large databases
- **Multi-Engine Support**: PostgreSQL and MySQL support
- **Flexible Storage**: Backup to local storage or S3-compatible cloud services
- **Automated Scheduling**: Built-in cron support for hands-off operations
- **Retention Policies**: Automatic cleanup of old backups to manage storage costs
- **Systemd Integration**: Run as a service with automatic resource management

## Quick Example

Here's a minimal configuration to get started:

```yaml
settings:
  binary:
    pg_dump: /usr/bin/pg_dump
  
  storages:
    local_backup:
      driver: local
      path: "/var/backups/postgresql"
      filename_prefix: "backup_"

backups:
  - name: "My Database"
    driver: postgresql
    connection:
      host: localhost
      port: 5432
      username: postgres
      password: your_password
      database: my_database
    storage:
      ref: local_backup
```

Run it:
```bash
dbackup backup -c backup.yml
```

That's it! Your database is backed up.
